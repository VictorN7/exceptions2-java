### Conhecimento aplicado:  

- Utilizada a estrutura do try-catch para os casos de tratamento de exceções ou propagação.
- Criação de Exception personalizada com a classe "DomainException" para testar exceções e compreender como aplicar para uma situação específica.
- Conhecer mais sobre os tipos de RuntimeException (não obriga a tratar exceções) e Exception (obriga a tratar exceções). 
- Utilização de métodos do tipo private para serem acessados somente pela classe.
- Boa prática nas delegações para cada método.
- Boa prática na separação dos códigos na classe onde está a regra de negócio.
